API Documentation

Endpoints available for integration

POST /api/verify

Verify a purchase code. Called by the Laravel application.

POST /api/verify Content-Type: application/json { "purchase_code": "12345678-1234-1234-1234-123456789012", "domain": "yourdomain.com", "product": "rocketlms" } // Success Response: { "valid": true, "license_type": "Regular license", "message": "License verified successfully." } // Error Response: { "valid": false, "error": "no_code", "message": "This purchase code is not registered." }

GET /api/licenses

List all registered licenses. Requires x-api-key header.

POST /api/licenses

Add a new license. Requires x-api-key header.

DELETE /api/licenses/:code

Delete a license by purchase code. Requires x-api-key header.

GET /api/health

Health check. No authentication required.
